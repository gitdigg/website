操作指南

本书是一本GitBook功能的实践项目，方便测试相关功能与插件。项目完全开源，需要请自取。

工具安装

$: npm i gitbook-cli -g

快速开始

# 样例项目
$: mkdir sample; cd sample
# 初始化目录
$: gitbook init
# 安装依赖
$: gitbook install
# 本地服务
$: gitbook serve
# 打包
$: gitbook build

项目下载

本项目地址: gitbook.可直接下载测试。

$: git clone https://github.com/liujianping/gitbook.git
$: cd guide
$: gitbook install
$: gitbook serve
