2.3 功能插件

这篇文章挺全的：GitBook 插件。

加两个简单插件，修改book.json文件相关配置块:

{
    ...
    "plugins": [
        "copy-code-button",
        "tbfed-pagefooter"
    ],
    "pluginsConfig": {
        "tbfed-pagefooter": {
            "copyright": "GitDiG.com @2019",
            "modify_label": "本文修订时间:",
            "modify_format": "YYYY-MM-DD HH:mm:ss"
        }
    }
    ...
}

增加新插件，需要执行插件安装命令:

$: gitbook install

再重新启动本地服务，查看效果。
